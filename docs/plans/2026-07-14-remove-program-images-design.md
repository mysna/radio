# 프로그램 이미지 제거 설계

## 목표

현재 재생 화면과 잠금 화면 미디어 정보에서 프로그램 이미지를 모두 제거한다.

## 범위

- 현재 재생 카드의 이미지 요소와 스타일을 제거한다.
- 화면 갱신 코드의 이미지 선택자 및 표시 처리를 제거한다.
- Media Session 메타데이터에 `artwork`를 넣지 않는다.
- 더 이상 사용하지 않는 EPG 이미지 URL 정규화 코드를 제거한다.
- 프로그램 제목, 채널명, 방송국·지역, 진행 바와 방송시간은 유지한다.

## 검증

정적 UI 테스트, EPG 정규화 테스트, Media Session 메타데이터 테스트를 이미지 없는 동작으로 갱신하고 전체 테스트를 실행한다.
