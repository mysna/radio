# 현재 재생 세로 이미지 배치 설계

## 목표

현재 재생 정보를 채널명, 방송국·지역, 프로그램 이미지, 프로그램명, 진행 바 순서의 단일 세로 흐름으로 표시한다.

## 레이아웃

- 기존의 이미지와 문구 2열 배치를 제거한다.
- 채널명과 방송국·지역을 먼저 표시하고, 프로그램 이미지를 그 아래에 둔다.
- 프로그램 이미지는 원본 비율을 보존한다. 가로형은 너비를 카드 폭에 맞추고, 세로형은 높이를 카드 폭에 맞춰 긴 변의 최대 크기를 카드 폭으로 제한한다.
- 프로그램명과 진행 바(시작·종료 시각 포함)는 이미지 다음에 표시한다.
- 이미지가 없는 경우 기존 동작처럼 이미지 영역을 숨겨 빈 공간을 만들지 않는다.

## 검증

정적 UI 테스트로 요소의 DOM 순서와 이미지의 비율 보존 및 긴 변 크기 제한을 검증하고 전체 테스트를 실행한다.
